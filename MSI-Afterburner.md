# To close `MSI Afterburner` to tray edit `MSIAfterburner.ini` in `C:\Program Files (x86)\MSI Afterburner`
* Field `SendToTrayOnClose		= 1`
# Do your setup of the GPU and save it as a profile (preferrably check online for the max clock speed of the GPU and make it as a profile)
# Setup your fan control from it afterwards and it will be saved even if there is a BSOD or power loss. Don't forget to enable the button `Apply at Windows Startup` (do this before locking profiles)
# Lock profiles to prevent unwanted edits.
# Open Settings and tick `Disable ULPS` to prevent crashing of the monitors (do this only if you have the issues)
