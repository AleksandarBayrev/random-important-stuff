# How to set up Minecraft
* Install `ATLauncher` (Flatpak)
* Login with Microsoft account
* Install your version of choice
* Add Fabric to Minecraft
* Download `FabricAPI`, `Iris`, `Iris Shaders`, `Sodium`, `FPSDisplay` mods for your version (or check `minecraft-mods` folder) (or restore the backup zip file with ATLauncher `Minecraft1204-2024_04_09_17_27_55.zip`)
* Play
